# AI Based Travel Planner

## Overview
The AI Based Travel Planner is a simple Flask-based web application that recommends travel destinations based on the user's budget. It also provides food recommendations, tourist attractions, best time to visit, and trip cost estimation.

---

# Features

- Destination recommendation based on budget
- Tourist place suggestions
- Food recommendations
- Best time to visit information
- Trip cost estimation
- REST API implementation using Flask
